# Ressources de la présentation sur les gestionnaires de mots de passe.

## Présentation générale sur le but du dépôt :

Ce dépôt contient les ressources liées à la présentation de l'outil "Gestionnaire de mots de passe" destinée à une classe de BTS SIO, en vue d'une utilisation dans les ateliers professionnels (AP).

L'objectif principal est d'assurer la continuité de service au sein des groupes de projet : si un étudiant est absent, ses collaborateurs doivent pouvoir accéder aux serveurs et aux comptes partagés grâce aux identifiants stockés de manière sécurisée dans le gestionnaire de mots de passe.

## Présentation du contenu du dépôt :

Ce dépôt met à disposition les documents suivants :

* **presentation_les-gestionnaires-de-mots-de-passe.pdf** : Le support visuel de la présentation détaillant les solutions Bitwarden, Passbolt et KeePass.
* **procedure_mise-en-place-gestionnaires-de-mots-de-passe.pdf** : Le guide étape par étape pour l'installation et la configuration (format PDF).
* **procedure_mise-en-place-gestionnaires-de-mots-de-passe.md** : Le guide étape par étape pour l'installation et la configuration (format Markdown).
